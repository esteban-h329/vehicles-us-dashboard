# vehicles-us-dashboard

Aplicación web para explorar un dataset de anuncios de venta de coches en EE.UU.

Construida con Streamlit y Plotly Express. Incluye un histograma del odómetro
y un gráfico de dispersión odómetro vs precio, activables con casillas de verificación.
El análisis exploratorio está en `notebooks/EDA.ipynb`.

**App en vivo:** https://vehicles-us-dashboard-jfuw.onrender.com
